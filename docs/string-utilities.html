字符串处理工具 V0.0.1

一个简单的字符串处理工具，使用 Python 语言开发，GUI 基于 Tkinter，主要包含功能：

1. MD5生成

2. Base64编码与解码

3. Base32编码与解码

源码地址：https://github.com/pythub-project/string-utilities

下载地址：Mac 端

Mac 端应用不能运行的，需要自行添加签名。执行如下命令

sudo codesign --force --deep --sign - 应用程序路径
