# 字符串处理工具

## 软件简介

一个简单的字符串处理工具，使用 Python 语言开发，GUI 基于 Tkinter，主要包含功能：

1. MD5 生成

2. Base64 编码与解码

3. Base32 编码与解码

源码地址：https://github.com/crogram-pythub/string-utilities

下载地址：[Mac 端](https://github.com/crogram-pythub/string-utilities/releases/latest)
